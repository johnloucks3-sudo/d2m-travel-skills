# D2M Travel Skills

Travel operations skills for luxury travel agencies — itinerary building, cruise pricing, client dossiers, invoice reconciliation, and portal emails.

**Author:** Dreams2Memories Travel, LLC  
**License:** MIT  
**Compatible with:** Claude Code 2.0+, Cursor, Gemini CLI, Goose, and any agent supporting the Claude Skills Specification

## Plugins

| Plugin | Description |
|---|---|
| `itinerary-builder` | Construct multi-day luxury trip itineraries from supplier data |
| `cruise-cost-compare` | Pull cabin prices from multiple cruise lines, compare and flag best value |
| `invoice-reconciler` | Match supplier invoices against bookings, flag discrepancies |
| `trip-dossier-generator` | Compile complete client trip dossier from booking data |
| `client-portal-emailer` | Draft personalized portal activation emails in D2M stationery |
